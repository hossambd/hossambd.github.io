Using Express

A common question is whether we need Express (or Node) to “connect” to the Java API.

We do not.

Express is a backend framework. If Java already owns authentication, adding Express would only introduce:

  • Extra latency
  • Duplicate logic
  • More infrastructure to maintain

in Express Our architecture would look like this:
